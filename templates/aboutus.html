How this works...?

This Web Application contains some fields to get the values from the user, we uses some fields like

  • Name
  • Register Number
  • Semester
  • Four Subject Names
  • Marks Obtained for that Subjects
  • Total Marks of that Subjects

If user forgotten to enter any field in this web application, it will notify to enter that field because all the fields in this web application are mandatory.
After getting Name,Register Number ,it will convert into uppercase for a standard certificate model and it directly displays in marklist with the Semester.
The Grade will calculate using some Criteria as follows:

Marks Grade
>=90 A
>=80 B
>=70 C
>=60 D
Below 60 F


The Semester Status like Passed/Failed is calculated using other method
If any one of the subject contain the Grade = F then the student will Failed in that Semester.
